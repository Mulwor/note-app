## Promise.all Polyfill

```js
const wait = (time) =>
  new Promise((resolve) =>
    setTimeout(() => {
      resolve(time);
    }, time)
  );

const arr = [wait(3000), wait(1000), wait(2000)];

const a = new Date();

Promise.all(arr).then((data) => {
  console.log(data);

  const diff = (new Date().getTime() - a.getTime()) / 1000;
  console.log(diff);
});

// --------

```

Работа с промисами. До решения кандидат также отвечает теорию про промисы.

### Решение:

Понимает устройство промисов, знает как работают статические методы. Код субъективен, поэтому оцениваем читабельность, скорость решения, кол-во подсказок, ход мыслей кандидата. Тут еще мб особенность языка, когда промисы резолвятся раньше времени. Кандидат должен быть способен продебажить свой код и найти решение. Задание на 20 мин максимум.
