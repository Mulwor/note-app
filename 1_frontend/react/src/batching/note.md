React Batching — это механизм, при котором React объединяет несколько вызовов setState в один процесс перерендера, чтобы избежать лишней работы и повысить производительность.

Простыми словами: если вы меняете состояние 10 раз подряд, React не перерисовывает компонент 10 раз. Он ждет, пока весь ваш синхронный код закончится, и делает всего один перерендер с финальным значением.
