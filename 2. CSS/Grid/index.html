CSS Grid Layout

Это удобная технология для раскладки элементов на веб-страницах. В отличие от flexbox, одновременно работающих только с одним измерением, grid дают возможность работать одновременно с двумя: горизонталью и вертикалью.

По умолчанию создает layout из одной колонки. Но если нам нужно использовать несколько колонок, то мы можем использовать grid-template-columns. display: grid; grid-template-columns: 25% 75%;

2

Передавая два значения 25% и 75%. Я говорю grid разделить эти элементы на два столбца

Столбцы могут быть определены с помощью любого действительного значения CSS , включая пиксели, rem, единицы измерения представления (vw, vh, vmin, vmax) и так далее.

Термин "fr" в CSS Grid обозначает "фракцию". Он является единицей измерения, позволяющей распределять пространство внутри контейнера сетки.

В примере, который вы предоставили, первая колонка установлена на 1 единицу пространства, а вторая колонка - на 3 единицы пространства. Это создает общую сумму в 4 единицы пространства.

Единица "fr" похожа на единицу "flex" в Flexbox, но она специально разработана для использования в CSS Grid. Она позволяет динамически распределять пространство внутри контейнера сетки.

Колонки, определенные с использованием единицы "fr", будут расти и сжиматься по мере необходимости, чтобы вместить содержимое. Это делает их полезными для создания гибких и адаптивных макетов.
